    Description An optional float  
    Remarks It is optional, initially it is not populated.
The property IsValidOrderedUnitPacksize indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidOrderedUnitPacksize = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

    Example 
DESADV_XSDLib.Line_Item_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION

// the IsValidOrderedUnitPacksize flag.

// So it can be set as follows

System.Diagnostics.Debug.Assert(oParent.IsValidOrderedUnitPacksize == true);
