Package_Reference
  Property Description  
    Property Name Package_Reference  
    Property Type  
    Parent Class DESADV_XSDLib.Line_Group  
    XML Type Element  
    Accessors Read Only  
    Description A collection of Package_References  
    Remarks Only one Element within this class may be populated at a time (this collection, as a whole is counted as an element).
This collection may contain 0 or 0 to Many entries.
Emptying the collection allows a different element to be the populated one.
If there is already a populated item, and an item is added to this collection then an exception will be raised
.

 
    Choice Properties Only one of the following properties may be populated at any one time
Line_Item
Line_Parties
Line_Measurements
Line_Order
Package_Reference

  
    Example 
DESADV_XSDLib.Package_Reference oItem;
DESADV_XSDLib.Line_Group oParent = ...;

// The collection object will already exist, but we may want to add items to it

// This method creates a new object adds it to the collection and 
// returns the new object
oItem = oParent.Package_Reference.Add();

// This way you can add an existing object to the collection
oItem = new DESADV_XSDLib.Package_Reference();
oParent.Package_Reference.Add(oItem);

//-------------------------------------------------------------
// To iterate through the collection

foreach(DESADV_XSDLib.Package_Reference itm in oParent.Package_Reference)
{
  System.Diagnostics.Debug.WriteLine(itm.[Some Property ....]);
}

// using an index
for(int i=0; i<oParent.Package_Reference.Count; i++)
{
  oItem = oParent.Package_Reference[i];
  System.Diagnostics.Debug.WriteLine(oItem.[Some Property ....]);
}
