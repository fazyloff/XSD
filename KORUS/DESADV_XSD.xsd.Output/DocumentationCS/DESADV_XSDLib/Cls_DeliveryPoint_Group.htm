If one of the elements is a collection, then the collection may contain a number of children. A collection is considered to be populated if it has any child items. A reference to a class is considered populated if it is not null. An enumeration or primitive has a corresponding XXXXValid function, if this is false it is not populated, true and it is. If an attempt is made to populate an element property while another one is already populated, then an exception is raised.
