LiquidTechnologies.Runtime.Net45.XmlDateTime DepartureDate
  Property Description  
    Property Name DepartureDate  
    Property Type LiquidTechnologies.Runtime.Net45.XmlDateTime  
    Parent Class DESADV_XSDLib.Transport_Group  
    XML Type Element  
    Accessors Read/Write  
    Description An optional LiquidTechnologies.Runtime.Net45.XmlDateTime  
    Remarks It is optional, initially it is not populated.
The property IsValidDepartureDate indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidDepartureDate = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

 
    Choice Properties Only one of the following properties may be populated at any one time
LocationName
DepartureDate
DepartureTime
ArrivalDate
ArrivalTime

  
    Also See IsValidDepartureDate - Indicates if the property has been populated.  
    Example 
DESADV_XSDLib.Transport_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidDepartureDate flag.
System.Diagnostics.Debug.Assert(oParent.IsValidDepartureDate == false);
      
// So it can be set as follows 
LiquidTechnologies.Runtime.Net45.XmlDateTime oNewItem = new LiquidTechnologies.Runtime.Net45.XmlDateTime();
oNewItem.SetDate(2005, 11, 29);
oParent.DepartureDate = oNewItem;
      
// As the property is populated IsValidDepartureDate will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidDepartureDate == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.DepartureDate.ToString());
