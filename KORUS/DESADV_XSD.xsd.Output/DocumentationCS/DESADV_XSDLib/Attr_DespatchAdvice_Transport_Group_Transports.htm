    Remarks It is optional, initially it is null.
In order to populate it a new DESADV_XSDLib.Transports object must be created and assigned to the property.
 
obj.Transports = new DESADV_XSDLib.Transports();
 
To 'un-populate' the property set it to null.
Only one Element within this class may be populated at a time, setting this property when another element is already populated will raise an exception. setting this property to null will allow another element to be selected
