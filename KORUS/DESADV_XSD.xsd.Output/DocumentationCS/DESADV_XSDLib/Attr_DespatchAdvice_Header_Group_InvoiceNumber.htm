    Description An optional string  
    Remarks It is optional, initially it is not populated.
The property IsValidInvoiceNumber indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidInvoiceNumber = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

    Example 
DESADV_XSDLib.DespatchAdvice_Header_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidInvoiceNumber flag.
System.Diagnostics.Debug.Assert(oParent.IsValidInvoiceNumber == false);
      
// So it can be set as follows
oParent.InvoiceNumber = "Sample Value";
      
// As the property is populated IsValidInvoiceNumber will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidInvoiceNumber == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.InvoiceNumber.ToString());
