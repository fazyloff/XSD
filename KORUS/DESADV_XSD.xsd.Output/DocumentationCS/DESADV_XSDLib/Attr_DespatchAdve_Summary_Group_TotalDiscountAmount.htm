float TotalDiscountAmount
  Property Description  
    Property Name TotalDiscountAmount  
    Property Type float  
    Parent Class DESADV_XSDLib.DespatchAdvice_Summary_Group  
    XML Type Element  
    Accessors Read/Write  
    Description An optional float  
    Remarks It is optional, initially it is not populated.
The property IsValidTotalDiscountAmount indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidTotalDiscountAmount = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

 
    Choice Properties Only one of the following properties may be populated at any one time
TotalAmount
TotalQuantity
TotalPackageNumber
TotalLines
TotalGrossAmount
TotalTaxAmount
TotalDiscountAmount
TotalFinalAmount

  
    Also See IsValidTotalDiscountAmount - Indicates if the property has been populated.  
    Example 
DESADV_XSDLib.DespatchAdvice_Summary_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidTotalDiscountAmount flag.
System.Diagnostics.Debug.Assert(oParent.IsValidTotalDiscountAmount == false);
      
// So it can be set as follows
oParent.TotalDiscountAmount = 1.245;
      
// As the property is populated IsValidTotalDiscountAmount will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidTotalDiscountAmount == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.TotalDiscountAmount.ToString());
