DESADV_XSDLib.Line_Group Class
This class represents a group of elements within the XML schema C:\BTS\XSD\KORUS\DESADV_XSD.xsd.
This class represents a choice. This means that only one of the elements within the class may be populated at a time. Elements are marked with an E in the table below. If one of the elements is a collection, then the collection may contain a number of children. A collection is considered to be populated if it has any child items. A reference to a class is considered populated if it is not null. An enumeration or primitive has a corresponding XXXXValid function, if this is false it is not populated, true and it is. If an attempt is made to populate an element property while another one is already populated, then an exception is raised.
