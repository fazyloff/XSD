string LocationName
  Property Description  
    Property Name LocationName  
    Property Type string  
    Parent Class DESADV_XSDLib.Transport_Group  
    XML Type Element  
    Accessors Read/Write  
    Description An optional string  
    Remarks It is optional, initially it is not populated.
The property IsValidLocationName indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidLocationName = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

 
    Choice Properties Only one of the following properties may be populated at any one time
LocationName
DepartureDate
DepartureTime
ArrivalDate
ArrivalTime

  
    Also See IsValidLocationName - Indicates if the property has been populated.  
    Example 
DESADV_XSDLib.Transport_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidLocationName flag.
System.Diagnostics.Debug.Assert(oParent.IsValidLocationName == false);
      
// So it can be set as follows
oParent.LocationName = "Sample Value";
      
// As the property is populated IsValidLocationName will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidLocationName == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.LocationName.ToString());
