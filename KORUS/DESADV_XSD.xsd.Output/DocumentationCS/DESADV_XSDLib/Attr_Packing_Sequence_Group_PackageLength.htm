float PackageLength
  Property Description  
    Property Name PackageLength  
    Property Type float  
    Parent Class DESADV_XSDLib.Packing_Sequence_Group  
    XML Type Element  
    Accessors Read/Write  
    Description An optional float  
    Remarks It is optional, initially it is not populated.
The property IsValidPackageLength indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidPackageLength = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

    Example 
DESADV_XSDLib.Packing_Sequence_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidPackageLength flag.
System.Diagnostics.Debug.Assert(oParent.IsValidPackageLength == false);
      
// So it can be set as follows
oParent.PackageLength = 1.245;
      
// As the property is populated IsValidPackageLength will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidPackageLength == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.PackageLength.ToString());
