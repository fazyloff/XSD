string AdditionalQuantity
  Property Description  
    Property Name AdditionalQuantity  
    Property Type string  
    Parent Class DESADV_XSDLib.Order_Measurement_Group  
    XML Type Element  
    Accessors Read/Write  
    Description An optional string  
    Remarks It is optional, initially it is not populated.
The property IsValidAdditionalQuantity indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidAdditionalQuantity = false.
Only one Element within this class may be set at a time, setting this property when another element is already populated will raise an exception.

 
    Choice Properties Only one of the following properties may be populated at any one time
AdditionalQuantity
AdditionalUnitOfMeasure

  
    Also See IsValidAdditionalQuantity - Indicates if the property has been populated.  
    Example 
DESADV_XSDLib.Order_Measurement_Group oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidAdditionalQuantity flag.
System.Diagnostics.Debug.Assert(oParent.IsValidAdditionalQuantity == false);
      
// So it can be set as follows
oParent.AdditionalQuantity = "Sample Value";
      
// As the property is populated IsValidAdditionalQuantity will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidAdditionalQuantity == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.AdditionalQuantity.ToString());
