Introduction
  What's in this help file?
  This help file contains documentation specific to the generated library.
This documentation is particularly useful when learning the structure of the generated class library, and for reference when working with the code.
It is not intended as a tutorial or getting started guide, for that see the main product documentation.
Note for the help file to work correctly the file LtHelp.chm should be in the same directory as this help file
  Where can I get additional help?
 

The main product help files contains the majority of the documentation, including

  • XML Data binding introduction (What is XML Data Binding)
  • documentation for the shared libraries (Re-distributable components)
  • Samples (XSD's, generated code, sample app's, and XML)
  • Trouble shooting sections
  • Getting started guide
