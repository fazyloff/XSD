[Element Primitive Type] PrimitiveValue
 PropertyDescription 
  Property NamePrimitiveValue  
  Property TypeStrongly Typed Primitive defined in Schema for Element (E.g. string, int, double)  
  AccessorsRead/Write 
  Description The textual data associated with the element 
  Remarks ie <e><b/>Test Text</e>
'Test Text' would the the Text for element e.

If there are a number of text blocks broken up by other elements then all the text is concatenated into a single string. ie <e>FirstText<b/>SecondText</e>
'FirstTextSecondText' would the the Text for element e.

When the text is written out is it always placed at the end of the element.
This behaviour is a known limitation of the libraries
